BIOTERM POINT

  • 물방울들은 꽉 차는 배경 이미지 위에
    수학과 관련된 math 메서드 스크립트를 짜서
    원들이 스스로 알아서 만들어지고
    커지면서 사라질 수 있게 해주었습니다

  • 스크롤을 할 경우
    클래스 이름을 붙여
    메뉴 색상을 변경해 주었습니다

  • 여러 장의 사진과
    과거에서 현재까지 이어지는 선,
    fade효과,
    작은 원에서 큰 원으로 변하는
    코드펜에서 배워 온 스크립트를 추가해
    재미를 주었습니다

  • 신제품이 나왔을 경우
    html 마크업에 텍스트와
    이미지만 추가하면 될 수 있도록
    display:flex 속성을 주어서
    차곡차곡 정렬되기 쉽게 하였습니다.

  • 제품 페이지에서는 img 버튼을 클릭할 경우
    이미지의 높이값,
    버튼의 텍스트,
    버튼의 색상을 변경할 수 있는 스크립트를 통해
    사용자가 데이터 비용을 절약할 수 있도록 도왔습니다.

  • 모바일 네비게이션의 경우
    키보드로 제어할 때 네비게이션 안에서
    처음과 마지막으로 돌아갈 수 있게끔
    data-link 속성을 추가하였습니다
